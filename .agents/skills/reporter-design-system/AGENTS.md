# reporter-design-system

## Overview

Design system and UI conventions for Reporter by Radiologic. Use this skill when creating or modifying any UI component, page, layout, menu, or interactive element.

## Structure

```
reporter-design-system/
  SKILL.md       # Main skill file - read this first
  AGENTS.md      # This navigation guide
  CLAUDE.md      # Symlink to AGENTS.md
  references/    # Detailed reference files
```

## Usage

1. Read `SKILL.md` for core principles and quick-reference table
2. Browse `references/` for detailed documentation on specific topics
3. Reference files are loaded on-demand -- read only what you need
